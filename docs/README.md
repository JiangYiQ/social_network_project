# 社交网络分析系统

## 一、项目概述

本项目为数据结构课程设计作品，基于图、哈希表、堆等自主实现的数据结构，开发社交网络图谱分析及智能推荐系统，支持人脉查询、社交距离计算、智能推荐等核心功能，旨在掌握数据结构底层原理与系统开发流程。

## 二、系统功能

### 基础功能
- **一度人脉查询**：查询指定用户的直接好友，采用邻接表直接查询，时间复杂度O(1)
- **二度人脉查询**：查询“好友的好友”，排除直接好友，采用BFS广度优先搜索加集合去重实现
- **社交距离计算**：计算两用户间最短路径，支持无权图和加权图两种模式。无权图使用BFS算法，加权图使用Dijkstra算法

### 扩展功能
- **多度人脉查询**：查询3度及以上人脉，可自定义度数，采用BFS层序遍历，记录每个节点的层数
- **兴趣推荐**：基于共同兴趣推荐非好友用户，采用集合交集运算计算共同兴趣数，按数量排序取前K个

### 辅助功能
- 数据清空与重新加载
- 结果导出为文本文件
- 打开数据目录
- 手动选择数据文件
